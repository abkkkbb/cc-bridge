# A÷笑传续作

那天下午，老豆又在客厅打盹，电扇吱呀吱呀地转，豆豆风从窗缝里挤进来，推着茶几上那本翻到一半的《肥宅养成图鉴》。
臭皮猪踩着螺旋桨从阳台飘进来，脚上还黏着一块昨天剩的炸鸡。"老豆老豆，我跟你说个秘密——我今天又被大蓝象给踩背了。"
老豆翻了个身，头顶"啵"地冒出一个水果糖。臭皮猪把水果糖抢过来塞进嘴里，咬了一口，"这糖怎么又是榴莲味。"
"榴莲是对的，"老豆含糊地说，"人生就是一颗榴莲糖，吃之前香，吃之后怀疑人生。"
正说着，电冰箱门突然自己打开，里面走出一只穿着西装的香蕉。香蕉鞠了个躬：
"请问，这里是宇宙答辩中心吗？"
老豆揉了揉眼睛："不是，是老豆家。隔壁第三家左转。"
香蕉"诶"了一声，转身又钻回冰箱，关门之前说了一句："那帮我告诉老豆，宇宙的答案是 42，但冰箱的答案是芒果。"
臭皮猪若有所思："芒果……莫非是老豆上周藏的那个？"
老豆猛地坐起："臭皮，那不是芒果，那是我老婆的青春。"

## 续续集

夜里两点，月亮是半颗荷包蛋。
大蓝象穿着睡衣溜到厨房，想偷偷煮一碗泡面。结果水还没烧开，电饭煲先说话了：
"大蓝象，你今天踩了几次老豆的背？"
"……三次吧，可能四次。"
"那你今晚的晚安故事就扣一半。"
大蓝象哭着说："那我只听半段总行吧？"
电饭煲叹了口气："好，给你讲半段：从前有一只……"
说到这里，它突然冒烟了，泡面也跟着开锅，汤汁溅出来浇在大蓝象的脚背上。
大蓝象大叫一声跳起来，把冰箱门撞开，那只香蕉又探出头："你们家到底是个什么中心？"
此时老豆从卧室走过来，头顶糖又换成了薄荷味。他平静地说："你们是不是有病？半夜三更的，宇宙答辩中心今天休息。"
然后把冰箱门关上，顺手给臭皮猪盖了条毛巾，回房间继续睡觉。

## 尾声

第二天早上，臭皮猪在沙发上醒来，发现螺旋桨被老豆收走了，茶几上放着一张纸条：
"臭皮：
螺旋桨没收一周，理由是昨晚飞得太低，把我的假发吹到电视后面去了。
—— 老豆"
臭皮猪气得嘴都歪了，决定去找大蓝象联盟报仇。
然而大蓝象正在被老豆用鸡毛掸子温柔地教育：
"踩背是可以的，踩心是不行的。"
大蓝象点头如捣蒜：
"明白了老豆，以后我就踩，不踩，心。"
